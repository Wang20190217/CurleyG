# Metasploit清除渗透痕迹

工具：Metasploit

为了实现这个目的，我们需要通过事件管理器模块清除事件日志。

这里，假设经过一系列的渗透测试已经获取了system权限的meterpreter，以下命令在meterpreter命令行下执行:

## 查看事件日志

```
run event_manager -i
```

## 删除事件日志

```
run event_manager -c
```

或者在meterpreter命令行中输入clearv命令清除目标系统的事件日志。
