# Metasploit渗透MSSQL

攻击机 kali 192.168.109.137

靶机 Win7_x64 192.168.109.139

数据库 MSSQL 2008 R2

MSSQL运行在TCP的1433端口以及UDP的1434端口

## 使用NMAP对MSSQL进行踩点

这里，我们使用Metasploit自带的db_nmap插件

**首先我们对目标的1433端口进行扫描**

```
db_nmap -sV -p 1433 192.168.109.139
```

