# WinlogonHack获取系统密码

## WinlogonHack截取密码的原理

WinlogonHack通过挂钩系统中的msgina.dll的WlxLoggedOutSAS函数记录登陆的账户密码。

当有3389脸上服务器时。新创建的winlogon.exe将会在登录前加载，注册了"Startup" 的DLLHook函数。登陆成功后，会将密码写入到.dat文件中去，并取消Hook。
