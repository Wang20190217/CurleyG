# 目录扫描神器DirBuster用法

不啰嗦，先上图：

![img](https://img-blog.csdnimg.cn/20190104095014139.jpg)

如果你扫描的目标是http://www.xxx.com/admin/，那么就要在URL to fuzz里填写"/admin/{dir}",意思是在"{dir}"的前后可以随意拼接你想要的目录或者后缀，例如输入": /admin/{dir}.php"就表示扫描admin目录下的所有php文件。
